# 金沙迷雾 · 古蜀秘案

一款基于古蜀金沙文明的互动推理解谜游戏。玩家扮演被神殿长老召唤的"外来者"，调查大祭司"残从"在密室中离奇死亡的真相。

## 在线体验

🎮 **GitHub Pages**：https://jingzhoujinga.github.io/bronze-mystery-game/

## 游戏简介

> 亥时三刻，金沙神殿净室门由内闩死，大祭司伏尸殿中，胸口插着一柄蜷蛇绿松石短剑。净室无窗，门由内反闩，守卫整夜值守却称未见一人——这是一场不可能的密室。

### 核心玩法

- **调查**：点击场景高亮区域搜查线索
- **审讯**：输入关键词或选择推荐话题与 5 位 NPC 对话
- **质疑**：用已收集线索戳穿角色的谎言，解锁隐藏信息
- **推理**：拖拽线索气泡到推理板，添加笔记并连线
- **断案**：提交档案卡，回答四道核心问题，揭晓真相

### 游戏阶段

| 阶段 | 内容 | 解锁条件 |
|------|------|----------|
| 阶段一 · 祭祀现场 | 神殿密室调查，与守卫、低阶祭司对话 | 游戏开始 |
| 阶段二 · 宫廷暗流 | 深入宫廷权力斗争，揭露人物关联 | 收集 ≥5 条线索 |
| 阶段三 · 水道迷踪 | 追踪地下水道线索，揭开最终真相 | 收集 ≥9 条线索 |

## 技术架构

- **纯前端 SPA**：原生 JavaScript，无框架依赖
- **四视图系统**：调查 / 审讯 / 线索 / 推理，单页切换
- **数据驱动**：叙事内容与游戏引擎完全解耦
- **本地存储**：localStorage 自动保存游戏进度

### 项目结构

```
web/
├── index.html          # 入口页面
├── css/
│   └── style.css       # 古蜀羊皮纸主题样式（32KB）
└── js/
    ├── app.js          # 游戏引擎主程序（1400+ 行）
    ├── data.js         # 阶段、场景、线索数据
    ├── data_chars.js   # 角色对话树 + 谎言系统
    └── data_extra.js   # 档案卡、攻略、推荐话题
```

### 核心算法

- **关键词匹配对话**：基于 `indexOf` 的多关键词迭代匹配
- **质疑谎言系统**：线索 ID 与谎言 challengeClueIds 比对验证
- **渐进式解锁**：线索收集数量驱动阶段推进
- **SVG 动态连线**：拖拽节点 + 贝塞尔曲线实时渲染
- **加权评分**：答题正确率 - 攻略惩罚（每次 -25 分）

## 本地运行

无需构建工具，直接用任意 HTTP 服务器启动：

```bash
# Python
cd web && python -m http.server 8080

# Node.js
npx serve web

# 或直接在浏览器中打开 web/index.html
```
